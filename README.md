# 🛒 Retail Sales Data Analysis

This project was developed during my technical internship to analyze retail transaction data. It focuses on data cleaning, processing, and deriving business insights using Python.

## 📁 Dataset
The analysis is based on a retail transaction dataset containing customer IDs, transaction dates, and amounts.

## 🛠️ Key Tasks Performed
* **Data Loading:** Handled encoding issues (ISO-8859-1) for seamless data reading.
* **Error Handling:** Implemented `try-except` blocks to manage FileNotFoundError.
* **Data Inspection:** Used `df.head()` and basic Pandas operations to understand data structure.

## 🚀 Libraries Used
* **Pandas:** For data manipulation and analysis.
* **OS:** For directory and path management.
